# Agent Space v3.4 Boundary

状态：v3.4 complete
目标：增加 thin Human Gateway。

## v3.4 新增

- Go gateway accepts `--human-port`。
- Human Gateway serves read-only HTTP:
  - `/`
  - `/api/audit`
  - `/artifacts/...`
- The HTML page shows:
  - audit entry count
  - event count
  - receipt count
  - receipts table
  - audit table
  - artifact links
- Node integration test verifies:
  - audit API returns entries
  - page renders the executed task and selected worker
  - artifact URL serves the Go-created artifact

## 为什么这是 v3.4

v3.3 made the Go runtime reachable over WebSocket。

v3.4 adds the first human-facing surface without turning the project into a chat/workspace product:

```text
Go execution
  -> audit log
  -> artifact file
  -> read-only Human Gateway
```

This is an operator window into the protocol proof。

## v3.4 不新增

- 不做 login/session/auth。
- 不做 approvals。
- 不做 task creation UI。
- 不做 chat。
- 不做 channels/threads/workspaces。
- 不做 admin tenant model。
- 不做 Octo clone。
