# Agent Space v10.3 Boundary

状态：complete
目标：Node can verify local artifact bytes and sidecar files against a receipt artifact manifest.

## v10.3 新增

- `verifyLocalArtifact(manifest)` verifies one local `artifact://local/...` artifact.
- The verifier reuses receipt manifest metadata checks.
- The verifier rejects sidecar JSON that differs from the receipt manifest.
- The verifier rejects local artifact bytes whose size or SHA-256 differs from the receipt manifest.

## 为什么这是 v10.3

v10.1 made Node write artifact manifests. v10.2 made Node receipt verification check signed manifest metadata.
v10.3 closes the local proof loop by checking the actual local bytes and sidecar file.

## v10.3 不新增

- 不做 HTTP artifact verify/read API。
- 不做 content-addressed mirror。
- 不做 artifact-store GC。
- 不做 object-store backend。
- 不做 remote artifact fetch。
- 不做 batch verifier。
