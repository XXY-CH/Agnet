# Agent Space v9.21 Boundary

状态：complete
目标：Node federation client can execute `FED_TASK_OPEN` against the Go federation gateway and verify the Go receipt.

## v9.21 新增

- `federation-gateway.mjs request` no longer asks for remote write scope by default.
- The existing Node federation client can send `FED_TASK_OPEN` to the Go summarizer worker.
- Node verifies the Go `FED_RECEIPT` signature through the returned worker descriptor and zone binding.
- Go integration coverage proves the Node request produces a Go receipt audit entry.

## 为什么这是 v9.21

Ultimate needs at least two implementations to expose protocol assumptions.
v9.21 turns the existing Node client and Go gateway into a real task-level interop proof instead of only resolve/query interop.

## v9.21 不新增

- 不做 Node server receiving Go client tasks。
- 不做 full conformance suite。
- 不做 public bind address。
- 不做 WebSocket TLS。
- 不做 Human Gateway HTTPS。
- 不做 QUIC。
- 不做 DHT / relay / edge gateway。
- 不做 A2A/ARD compatibility。
