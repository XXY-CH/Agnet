# Agent Space v11.13 Boundary

状态：complete
目标：Make the Node `FED_SWARM_CLOSE` verifier reject close proofs without a signed Swarm id.

## v11.13 新增

- Node `verifySwarmClose` requires the signed close body to include a non-empty `swarm_id`.
- The Node conformance test proves a trusted Zone signature is not enough when the close proof omits Swarm identity.

## 为什么这是 v11.13

`FED_SWARM_CLOSE` is supposed to bind a signed close proof to one Swarm. Before v11.13, a frame and close body that both omitted `swarm_id` could pass the frame/body equality check. v11.13 closes that identity-presence gap without adding a Node audit-backed completeness verifier.

## v11.13 不新增

- 不实现 Node audit-backed Swarm completeness verification。
- 不验证 step receipts against an audit log。
- 不实现 dynamic Swarm decomposition。
- 不实现 scheduler-owned routing。
- 不实现 parallel or cross-Zone Swarm execution。
- 不实现 public reachability proof。
- 不实现 A2A/ARD compatibility。
