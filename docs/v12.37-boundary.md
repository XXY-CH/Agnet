# Agent Space v12.37 Boundary

状态：complete
目标：Remove the upper-layer demo detour from the core proof-substrate line.

## v12.37 新增

- `upper-layer-demo.mjs` is removed from the core repository surface.
- `upper-layer-demo.test.mjs` is removed from the core verification suite.
- `docs/v12.36-boundary.md` is removed because the v12.36 upper-layer demo plan is no longer part of the core v12 line.
- README, ASP Core draft, implementation status, and v12 roadmap now present v12 as a proof-substrate/release-surface line again.

## 为什么这是 v12.37

The final upper layer will be developed separately. Keeping the 10-agent plus master-agent demo in this repository would pull v12 away from the Ultimate bottom substrate: verifier-owned proof bundles, receipt portability, transport evidence, package artifact proof, and eventually real external-host evidence. v12.37 records that correction as a versioned boundary instead of silently continuing the detour.

## v12.37 不新增

- 不实现 upper-layer demo, master-agent orchestration, or specialist-agent assignment execution。
- 不实现 automatic scheduler, semantic routing, ranking, or economy。
- 不实现 dynamic Swarm decomposition。
- 不实现 hosted public node。
- 不实现 real outside-host reachability。
- 不实现 package signing。
- 不实现 SBOM。
- 不实现 A2A/ARD compatibility。

This is a direction-correction slice for the core proof substrate, not a new orchestration runtime.
