# Agent Space v9.23 Boundary

状态：complete
目标：Go federation TCP listener can bind an explicitly configured host while keeping the default local-only posture.

## v9.23 新增

- `cmd/go-fed-discovery --listen-host <host>` configures the main federation TCP listener host.
- The default listen host remains `127.0.0.1`.
- `listenFederation` uses `net.JoinHostPort`, so configured hosts are not hand-concatenated.
- Go startup status and Human Gateway `/api/security` expose `listen_host` and derived `public_transport`.
- Go package coverage proves the configured loopback host is used, and public-host classification distinguishes loopback from non-loopback bind hosts.

## 为什么这是 v9.23

v9.21 and v9.22 closed bidirectional Node/Go task interop.
The next smallest transport step toward the Ultimate network layer is a configurable federation bind primitive, without claiming public deployment, NAT traversal, relay, or QUIC.

## v9.23 不新增

- 不改 WebSocket listener bind host。
- 不改 Human Gateway listener bind host。
- 不做 public gateway deployment。
- 不做 NAT / proxy / relay。
- 不做 QUIC。
- 不做 DHT / edge gateway。
- 不做 scheduler-driven cross-implementation dispatch。
- 不做 container namespace sandbox。
- 不做 A2A/ARD compatibility。
