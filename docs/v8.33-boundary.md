# Agent Space v8.33 Boundary

状态：complete
目标：Human Gateway page displays the current local approval session state.

## v8.33 新增

- Human Gateway page renders a `Session` panel.
- The panel calls the existing `GET /api/session` endpoint.
- A local bearer token can be entered for that read-only session check.
- The panel displays the derived approval actor and approval actions when authenticated.
- Missing or unknown tokens display an unauthenticated state.
- The Go integration test checks that the page includes the session UI and calls `/api/session`.

## 为什么这是 v8.33

v8.32 made local approval session state available as an API.
v8.33 adds the smallest browser-visible surface above that API:

```text
Human Gateway page
  -> local bearer token input
  -> GET /api/session
  -> approval actor/actions display
```

This lets a human see which local approval actor they are using without adding an account system.

## v8.33 不新增

- 不做 login form。
- 不做 password/session store。
- 不做 cookie session。
- 不做 roles/admin model。
- 不做 token storage or rotation。
- 不做 CSRF/session cookies。
- 不做 dynamic policy service。
- 不做 public deployment。
- 不做 A2A/ARD compatibility。
