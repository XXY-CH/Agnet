# Agent Space v8.44 Boundary

状态：complete
目标：Receipt-scoped artifact verify responses expose the receipt and audit proof digests they verified against.

## v8.44 新增

- `GET /api/artifacts/verify?task_id=<id>&uri=<artifact-uri>` returns `receipt_digest`.
- `GET /api/artifacts/verify?task_id=<id>&uri=<artifact-uri>` returns `audit_hash`.
- `receipt_digest` matches the signed receipt selected by task-scoped audit proof lookup.
- `audit_hash` matches the hash-chained audit entry containing that receipt.
- Existing artifact manifest verification still runs before returning success.
- The Go integration test checks both proof fields.

## 为什么这是 v8.44

v8.43 made task-scoped audit receipt proofs self-contained.
v8.44 makes artifact verification self-contained too:

```text
receipt task_id + artifact uri
  -> verified artifact manifest
  -> receipt_digest
  -> audit_hash
```

This gives clients a proof JSON path before they read artifact bytes.
