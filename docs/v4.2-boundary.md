# Agent Space v4.2 Boundary

状态：v4.2 complete
目标：给 signed merge proposal 增加只读 review surface。

## v4.2 新增

- Go CLI supports `--review-merges`。
- `--review-merges` verifies the audit log before listing merge proposals。
- Human Gateway exposes `/api/merge-proposals`。
- Human Gateway page shows a Merge Proposals table with:
  - task
  - worker
  - commit
  - changed files
- Node integration test verifies CLI review, API review, and page visibility。

## 为什么这是 v4.2

v4.1 proved signed checkpoint and merge proposal evidence。

v4.2 makes that evidence reviewable:

```text
audit log
  -> verify audit chain + receipt signatures
  -> list signed merge proposals
  -> expose proposals through Human Gateway
```

This is the smallest review surface before any promotion/apply path。

## v4.2 不新增

- 不自动 merge。
- 不做 apply button。
- 不做 conflict detection。
- 不做 branch queue。
- 不做 review auth。
- 不做 patch rendering。

## 验收

```bash
go test ./...
node --test --test-concurrency=1 *.test.mjs
```

The Go integration test must fail if a signed merge proposal is missing from CLI/API/page review surfaces。
