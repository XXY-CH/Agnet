# Agent Space v0.1 Boundary

状态：v0.1 candidate  
目标：在 v0 本地双进程闭环之上，冻结跨语言协议向量。

## v0.1 新增

- `test-vectors/asp-v0.json`
- 固定 Ed25519 seed
- 固定 `aid:` 输出
- 固定 canonical task
- 固定 task signature
- 固定 canonical receipt
- 固定 receipt signature
- Node 验证
- Go 标准库验证

## 为什么这是 v0.1

v0 证明 runtime 能跑。

v0.1 证明协议不是 Node 私有实现：

```text
same public key
  -> same aid
same canonical payload
  -> same Ed25519 signature verification
```

这为后续 Go runtime 或 Rust security components 铺路。

后续 descriptor 自签名边界见 `docs/v0.2-boundary.md`。

## v0.1 不新增

- 不重写 runtime。
- 不引入 Go 服务。
- 不改 frame 协议。
- 不新增 transport。
- 不做 federation。

## 验收

```bash
node --test --test-concurrency=1 *.test.mjs
go test ./...
```

必须通过。
