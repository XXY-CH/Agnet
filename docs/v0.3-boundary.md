# Agent Space v0.3 Boundary

状态：v0.3 candidate  
目标：让 audit log 从“可记录”升级为“可发现篡改”。

## v0.3 新增

- Hash-chained audit entries。
- 每条 audit line 包含：
  - `prev_hash`
  - `record`
  - `hash`
- `state/audit.head` 保存最新 hash。
- 新增 `verifyAuditEntries()` primitive。
- 新增 audit tamper detection 测试。

## 为什么这是 v0.3

v0.2 让 registry descriptor 可验证。

v0.3 让执行轨迹也开始可验证：

```text
change an old audit record
  -> later hash no longer matches
  -> verifier detects tampering
```

这仍然不是防篡改存储。拥有文件写权限的人仍可重写整个 log 和 head。v0.3 只证明 audit 格式支持 tamper-evident chaining。

后续 Zone-signed alias binding 边界见 `docs/v0.4-boundary.md`。

## v0.3 不新增

- 不做数据库。
- 不做 Merkle tree。
- 不做外部 timestamp。
- 不做 remote anchoring。
- 不做 blockchain。
- 不做 immutable storage。
