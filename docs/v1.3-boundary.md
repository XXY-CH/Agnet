# Agent Space v1.3 Boundary

状态：v1.3 candidate  
目标：增加最小跨 Zone capability discovery。

## v1.3 新增

- Agent Descriptor 增加 `capabilities`。
- Capabilities 纳入 Agent Descriptor 自签名内容。
- `FED_QUERY`
- `FED_QUERY_RESULT`
- `federation-gateway.mjs query`
- Exact capability match integration test。

## 为什么这是 v1.3

v1.2 只能按精确 `agent://` alias 获取远端 Agent。

v1.3 允许 Zone A 问 Zone B：

```text
谁有 summarize.text？
```

Zone B 返回匹配的 worker descriptor 和 Zone binding。Zone A 仍然必须验证：

- remote Zone trusted
- worker descriptor self-signature
- Zone binding
- capability claim is inside signed descriptor

## v1.3 不新增

- 不做向量搜索。
- 不做 semantic routing。
- 不做排序。
- 不做 reputation。
- 不做 multi-agent matching。
- 不做 remote registry sync。

这是 exact string match，不是智能发现网络。

后续 capability-to-task handoff 边界见 `docs/v1.4-boundary.md`。
