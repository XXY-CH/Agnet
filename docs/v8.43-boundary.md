# Agent Space v8.43 Boundary

状态：complete
目标：Task-scoped audit receipt proofs expose the selected receipt audit hash directly.

## v8.43 新增

- `GET /api/audit?task_id=<id>` returns `audit_hash`.
- `audit_hash` equals the hash-chained audit entry that contains the returned signed receipt.
- Existing task-scoped proof fields stay unchanged.
- The Go integration test checks the proof hash against the full audit entries.

## 为什么这是 v8.43

v8.42 exposed the receipt audit hash on artifact read responses.
v8.43 makes the task-scoped proof self-contained:

```text
receipt task_id
  -> task-scoped audit proof
  -> audit_hash
  -> signed receipt
```

This lets clients compare artifact read headers with the task proof without scanning the whole audit log.
