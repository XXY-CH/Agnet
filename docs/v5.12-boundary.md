# Agent Space v5.12 Boundary

状态：complete
目标：把 MCP `tools/call` arguments 摘要绑定进 sandbox evidence。

## v5.12 新增

- MCP sandbox evidence records `mcp_tool_arguments_digest`。
- Digest is computed from the exact arguments sent to `tools/call`。
- `sandbox_proof` signs this metadata because it signs sandbox evidence。

## 为什么这是 v5.12

v5.11 records selected tool schema lineage。

v5.12 records the actual argument payload identity:

```text
tools/call arguments
  -> argument digest
  -> sandbox evidence
  -> signed sandbox_proof
```

This prepares for argument policy without storing argument contents in the receipt。

## v5.12 不新增

- 不保存 arguments 明文。
- 不做 JSON Schema validation。
- 不做 policy engine。
- 不做 PII redaction。
- 不做 authorization UI。

## 验收

```bash
go test ./...
node --test --test-concurrency=1 *.test.mjs
```

The Go integration test must fail if MCP tool arguments digest is absent from sandbox evidence or not covered by the signed sandbox proof。
