# Agent Space v7.5 Boundary

状态：complete
目标：Human Gateway exposes explicit local queue actions without adding an automatic scheduler loop.

## v7.5 新增

- Human Gateway serves `GET /api/queue` from the durable queue directory.
- Human Gateway serves `POST /api/queue/actions` for:
  - `claim`
  - `drain`
- Claim action returns the lease id.
- Drain action reuses the existing queue drain path and records normal task audit/state evidence.
- Human Gateway page renders a read-only Queue table.

## 为什么这是 v7.5

v7.4 made retry/backoff state durable.

v7.5 gives humans and local tools a real queue control surface while keeping execution explicit:

```text
GET /api/queue
  -> inspect durable queue items

POST /api/queue/actions { action: "claim" }
  -> claimed + lease_id

POST /api/queue/actions { action: "drain" }
  -> running
  -> completed
```

## v7.5 不新增

- 不做 enqueue form。
- 不做 retry form。
- 不做 automatic scheduler loop。
- 不做 auth/login。
- 不做 multi-node scheduler。

## 验收

```bash
go test ./...
node --test --test-concurrency=1 test/go-fed-discovery.test.mjs
node --test --test-concurrency=1 test/*.test.mjs
```

The Go integration test must fail if `/api/queue` does not expose queued work, if Human Gateway claim does not return a lease id, or if Human Gateway drain does not complete the queued task through the normal queue execution path.
