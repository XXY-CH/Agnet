# Agent Space v12.1 Boundary

状态：complete
目标：Make the public proof bundle verifiable through one Node CLI command.

## v12.1 新增

- `asp-verify.mjs proof-bundle <bundle.json>` reads one public proof bundle manifest.
- The command delegates to the existing receipt artifact and Swarm close verifier paths.
- The command compares bundle `receipt_digest`, artifact URI/SHA-256/manifest-hash lists, signed `transport_proof`, and `swarm_close_digest` against verifier-owned outputs.
- The public proof integration test covers successful bundle verification and tampered `receipt_digest` rejection.

## 为什么这是 v12.1

v12.0 made the proof easier to consume by writing one manifest. v12.1 closes the next smallest gap: a consumer can now verify that manifest with one CLI command instead of manually running the receipt/artifact and Swarm close commands and comparing fields.

It does not add batch verification.

## v12.1 不新增

- 不实现 hosted public node。
- 不实现 external public reachability proof。
- 不实现 package publish/signing。
- 不实现 SBOM。
- 不实现 batch verification。
- 不实现 A2A/ARD compatibility。
- 不实现 scheduler-owned routing。
