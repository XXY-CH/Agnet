# Agent Space v8.36 Boundary

状态：complete
目标：Human Gateway exposes task-scoped audit receipt proofs.

## v8.36 新增

- `GET /api/audit?task_id=<id>` returns the existing receipt proof shape for one task.
- The endpoint reuses the existing `auditProof(task_id)` scan used by `FED_AUDIT_QUERY`.
- Human Gateway receipt table renders a `proof` link for each receipt task.
- The Go integration test checks both the task-scoped audit API response and the page proof link.

## 为什么这是 v8.36

The protocol already had task-scoped audit proof query over federation frames.
The Human Gateway only exposed full audit entries.
v8.36 connects the existing proof primitive to the local control plane:

```text
receipt task_id
  -> /api/audit?task_id=<id>
  -> receipt proof JSON
```

This makes receipt verification easier without building a broad audit viewer.

## v8.36 不新增

- 不做 audit search。
- 不做 audit diff/visualizer。
- 不做 pagination/filter UI。
- 不做 remote audit sync。
- 不做 distributed log search。
- 不做 roles/admin model。
- 不做 public deployment。
- 不做 A2A/ARD compatibility。
