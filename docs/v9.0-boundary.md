# Agent Space v9.0 Boundary

状态：complete
目标：artifact-store audit verification checks the mirror object index.

## v9.0 新增

- `--verify-audit --artifact-store <dir>` reads `<dir>/objects.ndjson`.
- Verification fails when the mirror index is missing.
- Verification fails when the mirror index does not include a receipt artifact manifest.
- The existing mirror byte and sidecar checks remain unchanged.

## 为什么这是 v9.0

v8.53 made filesystem mirrors enumerable through `objects.ndjson`.
v9.0 makes that index part of the audit proof before any larger artifact lifecycle work:

```text
receipt artifact manifest
  -> mirror bytes and sidecar
  -> objects.ndjson coverage check
```

## v9.0 不新增

- 不做 artifact GC。
- 不做 index compaction/dedup。
- 不做 remote object-store API。
- 不做 S3/MinIO backend。
- 不做 artifact browser。
- 不做 auth model for mirrored artifacts。
- 不做 container namespace sandbox。
- 不做 public deployment。
- 不做 A2A/ARD compatibility。

