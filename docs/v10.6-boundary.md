# Agent Space v10.6 Boundary

状态：complete
目标：Expose one-command local closure verification for a `FED_RECEIPT` frame and the local artifact bytes it names.

## v10.6 新增

- `asp-verify.mjs fed-receipt-artifacts <frame.json> <trusted-zones.json>` verifies one `FED_RECEIPT` frame.
- The command reuses `verifyFederatedReceipt`.
- The command verifies each local artifact manifest in the verified receipt with `verifyLocalArtifact`.
- The command rejects a receipt that names artifact refs but does not include artifact manifests.
- On success it prints `{"fed_receipt_artifacts_verify":"ok","task_id":"...","artifact_count":N}`.
- On failure it writes the verifier error to stderr and exits non-zero.

## 为什么这是 v10.6

v10.4 verified one local artifact manifest. v10.5 verified one `FED_RECEIPT` frame. v10.6 closes the smallest useful local delivery proof loop: one signed receipt plus the local artifact bytes it claims.

## v10.6 不新增

- 不做 npm package。
- 不做 batch receipt verification。
- 不做 receipt store/search。
- 不做 audit-log scan。
- 不做 remote artifact fetch。
- 不做 HTTP verifier service。
- 不做 object-store backend。
