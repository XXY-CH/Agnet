# Agent Space v10.1 Boundary

状态：complete
目标：Node artifact writes produce Go-shaped artifact manifest evidence.

## v10.1 新增

- Node `writeArtifact` writes `<artifact>.manifest.json`.
- The manifest includes `uri`, `sha256`, `size`, `media_type`, and `manifest_hash`.
- Node MVP, local runtime, and federation gateway include the manifest in `artifact.created` events.
- Node receipts include `artifact_manifests` alongside `artifact_refs`.
- The MVP test proves the sidecar matches the signed receipt manifest and that `sha256`, `size`, and `manifest_hash` match the artifact bytes.

## 为什么这是 v10.1

Go already had durable artifact manifest evidence. Node could create artifacts, but its receipts only carried artifact refs.
v10.1 closes that parity gap without introducing a Node artifact store or verifier.

## v10.1 不新增

- 不做 Node content-addressed mirror。
- 不做 Node artifact-store GC。
- 不做 Node artifact verify/read API。
- 不做 Node receipt verifier CLI。
- 不做 object-store backend。
- 不做 retention policy。
- 不做 scheduler。
