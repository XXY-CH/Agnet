# Agent Space v10.5 Boundary

状态：complete
目标：Expose Node `FED_RECEIPT` verification through the existing minimal CLI.

## v10.5 新增

- `asp-verify.mjs fed-receipt <frame.json> <trusted-zones.json>` verifies one `FED_RECEIPT` frame.
- The command reuses `verifyFederatedReceipt`.
- Trusted zones can be supplied as `{ "zones": [...] }` or as a raw array.
- On success it prints `{"fed_receipt_verify":"ok","task_id":"..."}`.
- On failure it writes the verifier error to stderr and exits non-zero.

## 为什么这是 v10.5

v10.2 strengthened the Node receipt verifier. v10.5 gives that verifier a command-line surface without publishing an npm package or creating a receipt store.

## v10.5 不新增

- 不做 npm package。
- 不做 batch verification。
- 不做 receipt store/search。
- 不做 audit-log scan。
- 不做 artifact byte fetch for remote receipts。
- 不做 HTTP verifier service。
