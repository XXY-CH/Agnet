# Agent Space v9.28 Boundary

状态：complete
目标：Swarm dependency evidence binds the upstream receipt digest, not only the upstream artifact manifest.

## v9.28 新增

- `FED_SWARM_OPEN` dependent step receipts now include `receipt_digest` in each `swarm.input_artifacts` entry.
- The digest is computed from the completed upstream signed receipt.
- `--verify-audit` checks `receipt_digest` against the upstream step receipt found earlier in the same audit.
- Integration coverage proves the live downstream Swarm receipt exposes the upstream receipt digest.
- Go package coverage proves a signed false upstream receipt digest is rejected.

## 为什么这是 v9.28

v9.27 made Swarm artifact dependency claims auditable.
v9.28 binds the dependency to the upstream receipt itself, which is the durable accountability object in the Ultimate design.

## v9.28 不新增

- 不做跨 audit Swarm proof lookup。
- 不做 remote audit sync。
- 不做 scheduler。
- 不做 dynamic decomposition。
- 不做 parallel DAG execution。
- 不做 conflict resolution。
- 不做 Swarm UI。
- 不做 Node Swarm server。
- 不做 A2A/ARD compatibility。
