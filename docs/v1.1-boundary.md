## 为什么这是 v1.1

```text
trusted Zone A -> trusted Zone B -> remote task receipt
```

```text
untrusted Zone A -> Zone B gateway -> FED_TASK_ERROR
```

后续 remote alias resolution 边界见 `docs/v1.2-boundary.md`。
