# Agent Space v8.34 Boundary

状态：complete
目标：Human Gateway receipt table exposes transcript artifacts that already exist in signed receipts.

## v8.34 新增

- Human Gateway receipt table now renders all `artifact_refs`, not only the first artifact.
- External/MCP `tool-transcript.json` artifacts are visible as local artifact links.
- Existing `/artifacts/...` static serving is reused for transcript artifact reads.
- The Go integration test checks that the Human Gateway page includes `tool-transcript.json`.

## 为什么这是 v8.34

v8.24 stored transcript bytes as signed local artifacts, but the Human Gateway page only displayed the summary artifact.
v8.34 closes that visibility gap with the smallest UI change:

```text
receipt.artifact_refs[]
  -> Human Gateway receipt table
  -> summary link + transcript link
```

This moves toward inspectable execution evidence without creating a separate transcript product.

## v8.34 不新增

- 不做 streamed transcript UI。
- 不做 transcript browser。
- 不做 artifact browser。
- 不做 remote transcript/object storage。
- 不做 object-store backend。
- 不做 artifact GC。
- 不做 auth model for artifact reads。
- 不做 container namespace sandbox。
- 不做 public deployment。
- 不做 A2A/ARD compatibility。
