# Agent Space v10.4 Boundary

状态：complete
目标：Expose the Node local artifact verifier through a minimal CLI.

## v10.4 新增

- `asp-verify.mjs artifact <manifest.json>` verifies one local artifact manifest file.
- On success it prints `{"artifact_verify":"ok","uri":"..."}`.
- On failure it writes the verifier error to stderr and exits non-zero.
- The MVP test proves the CLI accepts a valid manifest and rejects tampered local bytes.

## 为什么这是 v10.4

v10.3 added the library helper. v10.4 gives that proof a command-line surface without publishing a package or adding a service.

## v10.4 不新增

- 不做 npm package。
- 不做 receipt verifier CLI。
- 不做 HTTP artifact verify/read API。
- 不做 batch verification。
- 不做 remote artifact fetch。
- 不做 content-addressed mirror。
