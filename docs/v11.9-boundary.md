# Agent Space v11.9 Boundary

状态：complete
目标：Bind Node-to-Go interop receipt verification to the signed task sent by the Node client.

## v11.9 新增

- The Node `federation-gateway.mjs request` path passes its signed task into `verifyFederatedReceipt`.
- Node-to-Go interop now rejects returned receipts whose `task_digest` does not match the signed task the Node client sent.
- A Node integration test proves a signed remote receipt with a mismatched `task_digest` fails closed.

## 为什么这是 v11.9

v11.8 fixed the Go-to-Node direction. The Node-to-Go request path had the same in-memory signed task evidence but verified returned receipts without using it. v11.9 closes that symmetric interop gap without changing frame shapes or adding storage.

## v11.9 不新增

- 不改变 `FED_TASK_OPEN` frame shape。
- 不实现 task store/search。
- 不实现 remote task evidence retrieval。
- 不改变 audit query verification。
- 不实现 scheduler-owned routing。
- 不实现 dynamic Swarm decomposition。
- 不实现 A2A/ARD compatibility。
