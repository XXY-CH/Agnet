# Agent Space v9.16 Boundary

状态：complete
目标：Go sandbox probe records runtime binary digest evidence for configured container runtime candidates.

## v9.16 新增

- `--sandbox-probe container-namespace` records `runtime_sha256` when `AGNET_CONTAINER_RUNTIME` resolves to a readable command.
- `runtime_sha256` is the sha256 digest of the discovered runtime binary at `runtime_path`.
- `container-namespace` remains `supported: false` even when `runtime_sha256` is present.

## 为什么这是 v9.16

Ultimate sandboxing needs auditable runtime identity before trusting execution claims.
v9.16 fingerprints the configured runtime candidate without pretending that candidate has isolated any task.

## v9.16 不新增

- 不实现 container namespace sandbox。
- 不执行 Docker/OCI container tasks。
- 不做 Linux namespace setup。
- 不做 seccomp/AppArmor/profile enforcement。
- 不做 network namespace。
- 不做 filesystem mount namespace。
- 不做 cgroup limits。
- 不做 remote attestation。
- 不做 scheduler or automatic drain。
- 不做 A2A/ARD compatibility。
