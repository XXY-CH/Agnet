# Agent Space v5.11 Boundary

状态：complete
目标：把 selected MCP tool input schema 摘要绑定进 sandbox evidence。

## v5.11 新增

- If selected MCP tool descriptor includes `inputSchema`, sandbox evidence records `mcp_selected_tool_schema_digest`。
- `sandbox_proof` signs this metadata because it signs sandbox evidence。

## 为什么这是 v5.11

v5.10 proves the selected tool came from `tools/list`。

v5.11 records the selected tool's input shape:

```text
selected tool descriptor
  -> inputSchema digest
  -> sandbox evidence
  -> signed sandbox_proof
```

This prepares for argument policy without validating arguments yet。

## v5.11 不新增

- 不验证 tool arguments。
- 不保存完整 schema。
- 不做 JSON Schema validator。
- 不做 tool registry。
- 不做 authorization UI。

## 验收

```bash
go test ./...
node --test --test-concurrency=1 test/*.test.mjs
```

The Go integration test must fail if selected tool schema metadata is absent from sandbox evidence or not covered by the signed sandbox proof。
