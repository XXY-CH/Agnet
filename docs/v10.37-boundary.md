# Agent Space v10.37 Boundary

状态：complete
目标：Expose a narrow Node verifier command for portable public-listen Swarm close proof frames.

## v10.37 新增

- `asp-core.mjs` exports `verifySwarmClose(frame, trustedZones)`.
- `asp-verify.mjs swarm-close <frame.json> <trusted-zones.json>` verifies one `FED_SWARM_CLOSE` frame signature against trusted Zone material.
- The verifier prints `swarm_close_verify`, `swarm_id`, and the recomputed `swarm_close_digest`.
- `public-node-proof.test.mjs` verifies the generated public Swarm close frame and trusted Zone file through the CLI.
- `test-vectors.test.mjs` covers the Node verifier positive path and rejects a tampered close signature.

## v10.37 不新增

- 不做 full Swarm audit verifier CLI。
- 不做 close proof completeness/order/dependency verification in Node。
- 不做 Swarm receipt store/search。
- 不做 batch verification。
- 不做 HTTP verifier service。
- 不做新的 Swarm frame schema。
- 不做 dynamic Swarm decomposition。
- 不做 scheduler-owned DAG execution。
- 不做 parallel Swarm execution。
