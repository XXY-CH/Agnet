# Agent Space v12.3 Boundary

状态：complete
目标：Make public proof bundle proof-file paths resolve relative to the bundle manifest.

## v12.3 新增

- `scripts/public-node-proof.mjs` writes bundle proof-file references without the `state/` prefix.
- `asp-verify.mjs proof-bundle <bundle.json>` resolves relative proof-file paths relative to the bundle manifest file.
- `public-node-proof.test.mjs` asserts the bundle contains self-relative proof file paths.

## 为什么这是 v12.3

v12.2 made the public proof summary report bundle verification. v12.3 removes a small consumption leak: the bundle no longer hardcodes repo-root `state/...` proof-file paths for its receipt, trusted Zone, and Swarm close proof files.

## v12.3 不新增

- 不实现 artifact byte relocation。
- 不实现 hosted public node。
- 不实现 external public reachability proof。
- 不实现 package publish/signing。
- 不实现 SBOM。
- 不实现 batch verification。
- 不实现 A2A/ARD compatibility。
- 不实现 scheduler-owned routing。

It does not make artifact bytes relocatable.
