# Agent Space v11.8 Boundary

状态：complete
目标：Bind Go-to-Node interop receipt verification to the signed task sent by the Go client.

## v11.8 新增

- The Go `--interop-request` path keeps the signed task it sends to the Node gateway.
- Go interop receipt verification compares the returned receipt's `task_digest` against that signed task.
- Go tests prove mismatched interop task evidence fails closed with `receipt task_digest mismatch`.

## 为什么这是 v11.8

v11.7 exposed optional task evidence through the Go receipt CLI, but the Go-to-Node interop client already had stronger task evidence in memory and still verified the returned receipt without using it. v11.8 wires that existing evidence into the reusable receipt verifier.

## v11.8 不新增

- 不改变 `FED_TASK_OPEN` frame shape。
- 不实现 task store/search。
- 不实现 remote task evidence retrieval。
- 不实现 new interop protocol objects。
- 不实现 scheduler-owned routing。
- 不实现 dynamic Swarm decomposition。
- 不实现 A2A/ARD compatibility。
