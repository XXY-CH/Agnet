# Agent Space v9.22 Boundary

状态：complete
目标：Go client can execute `FED_TASK_OPEN` against the Node federation gateway and verify the Node receipt.

## v9.22 新增

- `cmd/go-fed-discovery --print-zone` prints the Go authority Zone descriptor for interop trust setup.
- `cmd/go-fed-discovery --interop-request <port>` sends one `FED_TASK_OPEN` request to a Node federation gateway.
- The Go client performs `HELLO` / `AUTH`, verifies the trusted Node Zone, verifies the returned zone binding, and verifies the Node `FED_RECEIPT` signature.
- Node integration coverage proves the Go request reaches the Node server and completes with `task.completed`.

## 为什么这是 v9.22

v9.21 proved Node client to Go task interop.
v9.22 closes the opposite direction with the smallest useful Go client proof, exposing protocol assumptions without building a full client SDK.

## v9.22 不新增

- 不做 full Go client SDK。
- 不做 full conformance suite。
- 不做 scheduler-driven cross-implementation dispatch。
- 不做 public bind address。
- 不做 WebSocket TLS。
- 不做 Human Gateway HTTPS。
- 不做 QUIC。
- 不做 DHT / relay / edge gateway。
- 不做 A2A/ARD compatibility。
