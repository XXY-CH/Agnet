# Agent Space v11.7 Boundary

状态：complete
目标：Expose optional task evidence checking through the Go receipt verifier CLI.

## v11.7 新增

- `go run ./cmd/go-fed-discovery --verify-receipt <receipt.json> --verify-task <task.json>` checks that the supplied task digest matches `receipt.task_digest`.
- Go receipt file verification rejects mismatched supplied task evidence with `receipt task_digest mismatch`.
- Existing `--verify-receipt` behavior is unchanged when `--verify-task` is absent.
- Audit verification paths keep using receipt-only verification.

## 为什么这是 v11.7

v11.5 added optional task evidence to the reusable Go verifier package, but the Go CLI still exposed only receipt-file verification. v11.7 connects the existing check to the CLI without introducing task storage, search, or remote retrieval.

## v11.7 不新增

- 不实现 task store/search。
- 不实现 remote task evidence retrieval。
- 不改变 audit verifier semantics。
- 不实现 batch verifier。
- 不实现 scheduler-owned routing。
- 不实现 dynamic Swarm decomposition。
- 不实现 A2A/ARD compatibility。
