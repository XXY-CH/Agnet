# Agent Space v12.2 Boundary

状态：complete
目标：Make the one-command public proof summary report proof bundle verification.

## v12.2 新增

- `scripts/public-node-proof.mjs` runs `asp-verify.mjs proof-bundle state/public-node-proof-bundle.json` after writing the bundle.
- The public proof summary returns `proof_bundle_verify: "ok"`.
- `test/public-node-proof.test.mjs` asserts the summary exposes that verifier result.

## 为什么这是 v12.2

v12.1 made the bundle verifiable by one CLI command. v12.2 makes the public proof itself report that command's result, so a consumer can see from the proof summary that the bundle has already passed the verifier path.

## v12.2 不新增

- 不实现 hosted public node。
- 不实现 external public reachability proof。
- 不实现 package publish/signing。
- 不实现 SBOM。
- 不实现 batch verification。
- 不实现 A2A/ARD compatibility。
- 不实现 scheduler-owned routing。

It does not add external public reachability proof.
