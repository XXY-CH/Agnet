# Agent Space v1.2 Boundary

状态：v1.2 candidate  
目标：让 Zone A 可以向可信 Zone B gateway 查询远端 `agent://` alias。

## v1.2 新增

- `FED_RESOLVE`
- `FED_RESOLVE_RESULT`
- `federation-gateway.mjs resolve`
- remote alias resolution integration test

## 为什么这是 v1.2

v1 证明跨 Zone task execution。

v1.1 证明 untrusted origin 会被拒绝。

v1.2 证明 Zone A 可以在发任务前获取并验证远端 Agent descriptor：

```text
Zone A trusts Zone B
Zone A asks Zone B for agent://zone-b/summarizer
Zone B returns worker descriptor + Zone binding
Zone A verifies remote Zone + binding + aid
```

## v1.2 不新增

- 不做 semantic routing。
- 不做 global discovery。
- 不做 remote registry sync。
- 不做 multi-agent capability search。
- 不做 caching。

这是精确 alias fetch，不是搜索网络。

后续 exact capability query 边界见 `docs/v1.3-boundary.md`。
