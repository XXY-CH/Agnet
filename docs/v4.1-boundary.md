# Agent Space v4.1 Boundary

状态：v4.1 complete
目标：给 artifact reference 增加 protocol-native manifest digest evidence。

## v4.1 新增

- Go artifact writes return an artifact manifest。
- `artifact.created` event carries `manifest`。
- Receipt records `artifact_manifests` alongside `artifact_refs`。
- Artifact manifest includes:
  - `uri`
  - `sha256`
  - `size`
  - `media_type`
  - `manifest_hash`
- Go audit verifier checks artifact ref / manifest alignment and manifest hash shape。
- Node integration test verifies artifact sha256, size, event manifest, receipt manifest, and audit verification。

## 为什么这是 v4.1

v4 proved protocol-native checkpoint evidence。

v4.1 strengthens artifact evidence:

```text
artifact.created
  -> artifact manifest
  -> receipt records manifest
  -> audit verifier checks manifest hash
```

This keeps artifact integrity in the Agent Space narrow waist without adding object storage。

## v4.1 不新增

- 不做对象存储。
- 不做远程 artifact 下载。
- 不做 artifact browser。
- 不做 retention policy。
- 不做 encryption。

## 验收

```bash
go test ./...
node --test --test-concurrency=1 test/*.test.mjs
```

The Go integration test must fail if artifact manifest evidence is missing from event stream, receipt, or audit verification。
