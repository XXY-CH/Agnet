# Agent Space v5.10 Boundary

状态：complete
目标：把 selected MCP tool 绑定到 `tools/list` evidence。

## v5.10 新增

- MCP adapter requires selected `tool_name` to appear in `tools/list`。
- Sandbox evidence records:
  - `mcp_selected_tool`
  - `mcp_selected_tool_digest`
- `sandbox_proof` signs this metadata because it signs sandbox evidence。

## 为什么这是 v5.10

v5.9 records MCP tool surface summary。

v5.10 proves the selected tool came from that surface:

```text
tools/list
  -> selected tool descriptor
  -> selected tool digest
  -> sandbox evidence
  -> signed sandbox_proof
```

This is the smallest tool-selection proof before schema or registry policy。

## v5.10 不新增

- 不保存完整 tools catalog。
- 不验证 input schema。
- 不做 tool registry。
- 不做 tool ranking。
- 不做 authorization UI。

## 验收

```bash
go test ./...
node --test --test-concurrency=1 *.test.mjs
```

The Go integration test must fail if the selected MCP tool is not bound into sandbox evidence or not covered by the signed sandbox proof。
