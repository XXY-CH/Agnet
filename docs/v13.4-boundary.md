# Agent Space v13.4 Boundary

状态：complete
目标：Land the first evidence-first semantic discovery/reputation ranking primitive without adding vector infrastructure.

## v13.4 新增

- Node federation gateway `FED_QUERY` may carry an `intent` string.
- Query results expose `discovery_evidence` for identity, exact capability match, semantic token match, trusted credential state, and receipt-count reputation input.
- Query results expose deterministic `ranking.score` and `ranking.reasons`.
- The exact capability candidate with trusted capability credential and receipt-count evidence ranks ahead of the semantic-only candidate.

## 硬边界

- The semantic-only candidate cannot outrank the exact capability candidate with trusted credential and receipt-count evidence.
- Ranking inputs are inspectable fields returned in the query result, not hidden vector distance.
- This is not global reputation, not vector-only routing, not public marketplace, and not Go query parity.

## v13.4 不新增

- No vector database, no global reputation coin, no public marketplace, and no Go query parity in this slice.
- 不实现 hosted/public reachability。
- 不实现 strong sandbox/remote attestation。
- 不实现 dynamic Swarm scheduling。
