# Agent Space v12.0 Boundary

状态：complete
目标：Make the local public-listen proof emit one verifier-facing proof bundle manifest.

## v12.0 新增

- `scripts/public-node-proof.mjs` writes `state/public-node-proof-bundle.json`.
- The bundle includes:
  - `receipt_frame`
  - `trusted_zones`
  - `receipt_digest`
  - verified artifact URI/SHA-256/manifest-hash lists
  - signed `transport_proof`
  - `swarm_close_frame`
  - `swarm_close_trusted_zones`
  - `swarm_close_digest`
- The public proof summary returns `bundle_manifest`.
- The focused test verifies the bundle matches existing verifier-owned outputs instead of recomputing a second proof path.

## 为什么这是 v12.0

v11 closed the trust-boundary bug line and signed transport evidence into public-listen receipts. v12 starts by making that proof easier to consume externally: one manifest points at the already verifier-ready files and digests. This is a release-surface step, not a new verifier or public network deployment.

## v12.0 不新增

- 不实现 hosted public node。
- 不实现 external public reachability proof。
- 不实现 package publish/signing。
- 不实现 SBOM。
- 不实现 batch verifier。
- 不实现 A2A/ARD compatibility。
- 不实现 scheduler-owned routing。
