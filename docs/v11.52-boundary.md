# Agent Space v11.52 Boundary

状态：complete
目标：Make Go filesystem artifact mirror index readers reject non-object `objects.ndjson` entries instead of preserving malformed rows.

## v11.52 新增

- Go `readArtifactStoreIndex` rejects `null` rows with `artifact mirror index invalid`.
- Focused Go test proves `objects.ndjson` containing `null` no longer reads as a valid nil map entry.
- The fix is scoped to mirror index row object presence; exact field matching remains owned by v11.51.

## 为什么这是 v11.52

The filesystem artifact mirror index is proof input for audit and receipt verification. JSON `null` decodes into a nil Go map without an unmarshal error, so the old reader could preserve a malformed row and still pass if another valid row matched the manifest. Mirror index readers should fail closed on non-object rows.
