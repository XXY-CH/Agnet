# Agent Space v8.41 Boundary

状态：complete
目标：Receipt-scoped artifact read responses expose the signed receipt digest.

## v8.41 新增

- `GET /api/artifacts/read?task_id=<id>&uri=<artifact-uri>` returns `X-Agent-Space-Receipt-Digest`.
- `HEAD /api/artifacts/read?task_id=<id>&uri=<artifact-uri>` returns the same header.
- The digest is computed from the signed receipt selected by task-scoped audit proof lookup.
- Existing receipt artifact verification still runs before returning headers or bytes.
- The Go integration test checks the header on both GET and HEAD responses.

## 为什么这是 v8.41

v8.40 made artifact proof headers available without bytes.
v8.41 binds those artifact responses to the complete signed receipt:

```text
receipt task_id + artifact uri
  -> signed receipt
  -> receipt digest header
  -> verified artifact headers/bytes
```

This is the smallest step from artifact proof toward full receipt-addressed artifact inspection.
