# Agent Space v12.21 Boundary

状态：complete
目标：Add a stable digest over the package proof manifest body before any future package signing or SBOM format is chosen.

## v12.21 新增

- `scripts/package-proof.mjs` computes `proof_digest` as `sha256(canonical(proof without proof_digest))`.
- The persisted `state/package-proof/package-proof.json` and stdout proof include the same `proof_digest`.
- `package-contract.test.mjs` verifies `proof_digest` is 64 lowercase hex and recomputes it from the package proof body.

## 为什么这是 v12.21

v12.20 persisted the package proof as a stable JSON file, but future signing or SBOM work still needed a digest of that proof content. v12.21 adds a stable digest over the package proof manifest body while keeping the release surface local and unsigned.
