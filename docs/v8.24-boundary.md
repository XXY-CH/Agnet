# Agent Space v8.24 Boundary

状态：complete
目标：External/MCP tool result transcripts are persisted as local artifacts.

## v8.24 新增

- External/MCP tools write `artifact://local/<task_id>/tool-transcript.json`.
- The receipt includes the transcript artifact ref and manifest alongside the summary artifact.
- Sandbox evidence records `tool_transcript_ref` next to the existing `tool_transcript_digest`.
- The Go integration test reads the transcript artifact and verifies its SHA-256 equals `tool_transcript_digest`.

## 为什么这是 v8.24

v8.18 recorded transcript digests, but did not keep the transcript bytes.
v8.19-v8.23 made local artifacts durable and verifiable.

v8.24 connects those pieces:

```text
tool transcript bytes
  -> tool_transcript_digest
  -> local artifact + manifest
  -> signed receipt artifact refs
  -> existing audit artifact verification
```

## v8.24 不新增

- 不做 streamed transcript UI。
- 不做 transcript browser。
- 不做 remote transcript/object storage。
- 不做 artifact GC。
- 不做 long-running MCP session reuse。
- 不做 container sandbox。
- 不做 public deployment。
- 不做 A2A/ARD compatibility。

## 验收

```bash
go test ./...
node --test --test-concurrency=1 test/go-fed-discovery.test.mjs
node --test --test-concurrency=1 test/*.test.mjs
```

The Go integration test must fail if an external/MCP receipt lacks a local transcript artifact whose bytes match `sandbox.tool_transcript_digest`.
