# Agent Space v8.18 Boundary

状态：complete
目标：External/MCP sandbox evidence records a tool result transcript digest without storing the full transcript.

## v8.18 新增

- External sandbox evidence records `tool_transcript_digest` as SHA-256 over raw stdout bytes.
- MCP sandbox evidence records `tool_transcript_digest` as SHA-256 over the final `tools/call` response object.
- `sandbox_proof` signs the same sandbox evidence, so the transcript digest is covered by the Zone authority signature.

## 为什么这是 v8.18

Ultimate execution evidence needs to identify not only which command and binary ran, but also which tool result transcript was observed.

v8.18 keeps the smallest privacy-preserving version:

```text
tool result transcript
  -> tool_transcript_digest
  -> sandbox evidence
  -> signed sandbox_proof
```

## v8.18 不新增

- 不存完整 transcript。
- 不做 streamed output UI。
- 不做 transcript artifact store。
- 不做 long-running MCP session reuse。
- 不做 container sandbox。
- 不做 public deployment。
- 不做 A2A/ARD compatibility。
