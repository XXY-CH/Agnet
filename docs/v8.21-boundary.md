# Agent Space v8.21 Boundary

状态：complete
目标：Audit verification checks local artifact bytes against signed artifact manifests.

## v8.21 新增

- `--verify-audit` reads each local `artifact://local/...` file referenced by a receipt.
- The verifier rejects artifacts whose byte length differs from the manifest `size`.
- The verifier rejects artifacts whose SHA-256 digest differs from the manifest `sha256`.
- The Go integration test tampers with an artifact after a successful audit verify and expects `--verify-audit` to fail.

## 为什么这是 v8.21

Ultimate artifact evidence is not enough if the receipt only proves that a manifest was internally consistent.

v8.21 closes the local verification loop:

```text
receipt artifact manifest
  -> local artifact bytes
  -> size + sha256 check
  -> audit verification result
```

## v8.21 不新增

- 不做 remote artifact fetch。
- 不校验 remote object store。
- 不做 artifact GC。
- 不做 artifact browser。
- 不做 full transcript artifact storage。
- 不做 container sandbox。
- 不做 public deployment。
- 不做 A2A/ARD compatibility。

## 验收

```bash
go test ./...
node --test --test-concurrency=1 go-fed-discovery.test.mjs
node --test --test-concurrency=1 *.test.mjs
```

The Go integration test must fail if `--verify-audit` accepts a receipt whose local artifact bytes no longer match the manifest.
